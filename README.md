# 🌊 Water Quality & Potability Prediction

Machine Learning project focused on predicting water potability using physicochemical water quality indicators.

This project explores how data science and machine learning can support environmental monitoring and public health by identifying whether water is safe for human consumption.

---

# 📌 Project Overview

Access to safe drinking water is one of the most important public health challenges worldwide.

In this project, I analyze a public water quality dataset and build machine learning models capable of predicting water potability based on several chemical and physical properties.

The project covers the complete data science workflow:

* Data cleaning
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine learning modeling
* Model evaluation
* Insights and interpretation

---

# 🎯 Objectives

The main objectives of this project are:

* Analyze water quality indicators and their distributions
* Identify relationships between physicochemical variables
* Handle missing values and prepare data for modeling
* Train and compare multiple machine learning models
* Evaluate model performance using classification metrics
* Understand which features influence water potability the most

---

# 🧪 Dataset Information

The dataset used in this project is publicly available on Kaggle.

It contains water quality metrics that help determine whether water is potable (safe to drink) or not.

## Variables Included

| Feature         | Description                                    |
| --------------- | ---------------------------------------------- |
| pH              | pH value of water                              |
| Hardness        | Capacity of water to precipitate soap          |
| Solids          | Total dissolved solids                         |
| Chloramines     | Amount of chloramines                          |
| Sulfate         | Amount of sulfates dissolved                   |
| Conductivity    | Electrical conductivity of water               |
| Organic Carbon  | Amount of organic carbon                       |
| Trihalomethanes | Amount of trihalomethanes                      |
| Turbidity       | Measure of water cloudiness                    |
| Potability      | Target variable (1 = potable, 0 = not potable) |
